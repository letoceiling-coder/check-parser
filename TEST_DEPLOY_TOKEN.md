# Проверка DEPLOY_TOKEN на сервере

## Выполните на сервере для проверки:

```bash
cd ~/project.siteaccess.ru/public_html

# Проверить, что токен читается через tinker
php artisan tinker --execute="echo env('DEPLOY_TOKEN');"

# Должен вывести: bedbae66b3e1288f8d5fb6c40dc03295b13f5838e8d90c2d0952b81555047ad4
```

## Если токен читается правильно:

Теперь можно выполнить `php artisan deploy` локально, и deploy должен работать.

## Если все настроено правильно:

После выполнения `php artisan deploy` локально, сервер должен:
1. Обновить код из Git
2. Установить зависимости Composer
3. Выполнить миграции
4. Очистить кеши
5. Вернуть успешный ответ
